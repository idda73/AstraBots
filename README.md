\## Overview

AstraBots is a multi-agent robotics orchestration and validation platform designed to simulate coordination conflicts before real-world deployment.



\## Features

\- Multi-robot task allocation

\- Conflict detection and reassignment

\- Live warehouse simulation

\- Performance metrics (efficiency, runtime, conflicts)

\- Web-based control dashboard



\## Architecture

\- FastAPI backend

\- Threaded simulation engine

\- REST API endpoints

\- HTML5 Canvas simulation frontend

\- Designed for Vultr VM deployment



\## Setup Instructions



\### 1. Clone repository

git clone <repo-url>



\### 2. Install dependencies

pip install -r requirements.txt



\### 3. Run server

python -m uvicorn app:app --reload



Open http://127.0.0.1:8000
